# Reactor Core Clojure

A Clojure(Script) library for reactor-core.

## Usage

```
[io.tsachev/reactor-core-clojure "0.1.0-SNAPSHOT"]
```

```clojure
(refer-clojure :exclude [concat empty filter map range reduce take])
(use 'reactor-core)
```
